Operating Systems Terminology

TERM DEFINITION
operating system The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue Organized on a list, jobs are in queue waiting to be processed in a first-in, first-out fashion.
FIFO FIFO stands for first-in, first-out. See job queue.
interactive processing Development that allows a program that is being executed to carry on a dialogue through remote terminals.
time-sharing multiprocessing Implemented a job queue to provide service for multiple users at what seems like the same time, but dividing the execution of the job in intervals.
application software Consists of the programs for performing tasks particular to the machine's utilization.
system software Performs those tasks that are common to computer systems in general. Provides the infrastructure that the application software requires.
utility software Consists of software units that extend the capabilities of the operating system.
graphical user interface (GUI) Modern systems use this so objects can be manipulated, such as files and programs, and are represented pictorially on the display as icons.
shell An old user interface that communicated with users through textual messages using a keyboard and monitor screen.
kernel The internal part of an operating system.
device driver Software units that communicate with the controllers (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine.
file manager Software unit whose job is to coordinate the use of the machine's mass storage facilities. Maintains records of all the files store in mass storage, including where each file is located, which users are allowed to access the various files, and which portions of mass storage are available for new files or extensions to existing files.
memory manager Charged with the task of coordinating the machine's use of main memory.
virtual memory Large "fictional" memory space created by paging.
boot strapping A procedure that takes place every time a computer is turned on. It transfers the operating system from mass storage into main memory.
ROM A type of computer storage that contains non-volatile, permanent data that can only be read, not altered. Stands for "read-only memory."
deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
spooling The technique of holding data for output at a later but more convenient time.
super user (or administrator) Highly privileged access to the operating system. Administrator can alter settings within the operating system, modify critical software packages, adjust the privileges granted to other users, and perform a variety of other maintenance activities that are denied normal users.
multi-factor authentication An authentication method that requires the user to provide two or more verification factors to gain access to a resource such as an application, online account, or VPN. A growing number of online accounts require two-factor authentication.
