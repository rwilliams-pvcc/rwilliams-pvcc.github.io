Programming Languages

TERM DEFINITION
machine language Program sequences of instructions that are encoded as numeric digits, such an encoding system is known as a machine language.
debugging The process of locating and correcting errors in programs in a machine language before the job is finished.
assembler Program created to convert mnemonic expressions into machine language instructions.
assembly language A mnemonic system for representing programs. Became known as second-generation languages due to how revolutionary assembly language was.
translator A program that translates programs expressed in high-level primitives into machine-language programs.
compiler A translation program that had to often compile several machine instructions into short sequences to simulate the activity requested byu a single high-level primitive.
interpreter An alternative to translators, interpreters emerged as another means of implementing third-party languages. They executed the instructions as they were translated instead of recording the translated version for future use.
cross-platform software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network.
natural language Languages such as English, German, and Latin and evolved in grammars naturally.
formal language Programming languages that evolve over time without formal grammatical analysis.
machine independence A language that does not rely on the characteristics of a particular machine, making it independent.
high-level programming language Development of higher level programming languages raises the level at which programmers conceptualize software designs and enabling the construction of increasingly complex software systems. Higher-level programming languages make computer programs both easier for human authors to write, and easier for subsequent readers to understand.
object-oriented programming (OOP) A programming process where a software system is viewed as a collection of units, called objects, and is capable of performing the actions that are immediately related to itself as well as requesting actions of other objects. This programming process goes with the programming paradigm called object-oriented paradigm.
scripting language A collection of languages that is a subset of the imperative programming languages. These languages are typically used to perform administrative tasks rather than develop complex programs.
data types:
  • integer
  • float
  • string
  • Refers to numeric data consisting of whole numbers, probably stored using two's complement notation.
  • Refers to numeric data that might contain values other than whole numbers, probably stored in floating-point notation.
  • Refers to data that is in a sequence of characters.
primitive data type Data types that are included as primitives in a programming language. This includes integer, float, character and Boolean.
data structure The conceptual shape or arrangement of data.
Python collection data structures
  • list
  • tuple
  • set
  • dictionary
  • Used to store multiple items in a single variable. Created using square brackets and are ordered, changeable, and allow duplicate values.
  • Used to store multiple items in a single variable. Created using round brackets and is a collection which is ordered and unchangeable.
  • Used to store multiple items in a single variable. Created using curly brackets and is a collection which is unordered, unchangeable, and unindexed.
  • Used to store data values in key:value pairs. Created using curly brackets and have keys and values and is a collection which is ordered, changeable, and does not allow duplicates.
function A set of instructions for performing a task that can be used as an abstract tool by other program units.
local variable A variable declared within a function and can be referenced only within that function.
global variable Variables whose scopes are not restricted to a particular part of a program.
source program A program in the original form.
object program The translated version of the source program.
